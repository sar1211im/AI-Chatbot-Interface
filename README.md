# AI Chatbot using React.js & Google Gemini API

## 📌 Project Overview

AI Chatbot is a modern web-based conversational AI application built using React.js and powered by Google's Gemini API. The application enables users to interact with an AI assistant through a clean and responsive chat interface, receiving real-time responses to questions and prompts.

This project demonstrates the integration of Generative AI with modern frontend development technologies and showcases concepts such as Prompt Engineering, API Integration, and Conversational AI.

---

## 🚀 Features

* Real-time AI-powered conversations
* Google Gemini API integration
* Responsive chat interface
* Markdown support for AI responses
* Auto-scrolling chat history
* Loading indicator during response generation
* Clean and modern UI
* Mobile-friendly design

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/chat-ai.git
cd chat-ai
```

---

### Install Dependencies

```bash
npm install
```

---

## 🔑 Configure Gemini API Key

Create a `.env` file in the root directory:

```env
VITE_API_GENERATIVE_LANGUAGE_CLIENT=YOUR_GEMINI_API_KEY
```

Replace:

```text
YOUR_GEMINI_API_KEY
```

with your actual Gemini API key.

---

## ▶️ Run the Application

```bash
npm run dev
```

Application will start at:

```text
http://localhost:5173
```

---

## 🔍 How It Works

1. User enters a prompt in the chat box.
2. React sends the request to the Gemini API.
3. Gemini processes the prompt.
4. AI-generated response is returned.
5. Response is displayed in the chat window.
6. Conversation history is maintained during the session.
